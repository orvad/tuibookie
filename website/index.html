The story

Why TuiBookie?

Tired of scrolling through shell history to find that one SSH command? TuiBookie was born from a simple frustration: too many servers, too many commands, no good way to organize them in the terminal.

Built originally as an SSH bookmark manager, TuiBookie grew into a general-purpose CLI command organizer. Save any command — SSH connections, Docker workflows, kubectl operations, deployment scripts — and run them instantly from a clean, keyboard-driven interface.

No browser. No GUI. Just your terminal, organized.

Features

Everything you need, nothing you don't.

[:]

Categories

Group commands by project, environment, or workflow. Add, rename, delete on the fly.

⇄

Import / Export

Back up your bookmarks as JSON. Import from file picker or manual path.

hjkl

Vim Navigation

Arrow keys and vim-style h/j/k/l navigation. Drill in with →, back with ←.

>_

Execute Instantly

Hit Enter to run any bookmarked command directly from the TUI.

~/.cfg

Configurable

Change your bookmarks file path right from Settings. Or use --config flag and env var for scripting.

■■■■■

Monokai Theme

Beautiful dark color scheme with pink accents and green highlights. Easy on the eyes.

Get started

brew tap orvad/tuibookie && brew install tuibookie

go install github.com/orvad/tuibookie@latest
